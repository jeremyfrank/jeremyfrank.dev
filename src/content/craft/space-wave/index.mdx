SVG path animation emulating a star traveling along a cylindrical sign wave path. The `.getTotalLength()` method is used to determine `stroke-dasharray` values, while the `stroke-dashoffset` value is calculated with a distance per frame constant using `requestAnimationFrame()`. Currently implemented on [Viget Explorations](https://explorations.viget.com/).
